# EngCourseApplyBot

**EngCourseApplyBot** — Telegram-бот для записи на курсы английского языка и определения уровня владения английским с помощью AI.

Бот помогает пользователю:

- определить уровень английского
- узнать информацию о курсах
- оставить заявку на обучение
- отправить контактные данные администратору

Бот написан на **Python** с использованием **aiogram 3**, **FSM** и интеграции с **AI API** для анализа текста.

---

## Возможности

Бот поддерживает следующие функции:

- 🤖 **AI-оценка уровня английского**
- ✍️ **Анализ текста пользователя**
- 📊 **Определение уровня по шкале CEFR (A1–C2)**
- 📝 **Оставление заявки на курс**
- 📞 **Сбор контактных данных пользователя**
- 📩 **Отправка заявки администратору**
- 🔄 **FSM-логика диалога**
- 📘 **Информация о курсе**

Пользователь может написать текст о себе на английском, после чего бот проанализирует его и определит примерный уровень владения языком.

## Установка
- Клонировать репозиторий
```
git clone https://github.com/boomred8/EngCourseApplyBot.git
cd EngCourseApplyBot
```
- Установить зависимости
```
pip install -r requirements.txt
```
- Создать .env файл 
```
BOT_TOKEN=your_telegram_bot_token
USER_ID=telegram_admin_id

OPENAI_API_KEY=your_openai_key
HF_API_KEY=your_huggingface_key
HF_API_URL=api_url
```
Токены и ключи загружаются через python-dotenv. 
- Запустить бота
```
python run.py
```
После запуска бот начинает получать сообщения через polling.

## Как работает бот
1.  Пользователь запускает бота командой: ``` /start ```
2.  Бот предлагает:
   - проверить уровень английского.
   - узнать информацию о курсе.
3.  Пользователь пишет текст о себе на английском.
4.  AI анализирует текст и определяет уровень.
5.  Пользователь может оставить заявку на курс.
6.  Бот отправляет заявку администратору.

Заявка содержит:
- имя пользователя
- номер телефона
- username Telegram
- Telegram ID
Обработка заявки реализована через ```FSM```.
